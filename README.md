# Rest Countries

## Overview

This project is a web application that lets you explore information about countries around the world. It helps you quickly find details like population, region, capital, and even bordering countries, making it easy to learn more about different nations.

## Features

- **View All Countries**: See a comprehensive list of countries on the homepage.
- **Search for Countries**: Easily find specific countries using a search bar.
- **Filter by Region**: Narrow down the list of countries by continent or region.
- **Dark Mode Toggle**: Switch between light and dark themes for comfortable viewing.
- **Country Details Page**: Click on any country to view in-depth information, including native name, population, currencies, languages, and more.
- **Border Country Navigation**: Seamlessly navigate to the details of bordering countries.
- **Pagination**: Browse through large lists of countries with intuitive pagination controls.

## Usage

The application presents a list of countries upon loading.

1.  **Search**: Use the search bar at the top to type in a country name. The list will update dynamically as you type.
2.  **Filter**: Use the "Filter by Region" dropdown to select a specific region (e.g., Africa, Asia).
3.  **View Details**: Click on any country card to navigate to its dedicated details page.
4.  **Navigate Borders**: On the country details page, you can click on any listed border country to view its details.
5.  **Go Back**: Use the "Back" button on a country details page to return to the previous view.
6.  **Theme Toggle**: Use the "Dark Mode" button in the header to switch the application's theme.
7.  **Pagination**: If there are many countries matching your search/filter, use the "Next", "Prev", and "Home" buttons at the bottom to navigate through pages.
